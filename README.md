# house-price-data-modelling-
The following data set house. CVS provides housing market data for 20640 houses from California and includes following features like median household income, population, total rooms, median house age, and proximity to the ocean, with the target variable being median house value. 
The goal of my research is: - 
1)	Develop and check accuracy of linear regression model 
2)	Tune its hyperparameters and compare the difference 
3)	Develop a KNN model 
4)	Find how changing K affects overall performance 
5)	Predict house price using both models and make a comparison 
6)	Discuss limitation and advantages of both the models
